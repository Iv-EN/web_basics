## Локальная установка приложения
1. Клонируйте репозиторий:
```bash
git clone https://github.com/Iv-EN/web_basics.git
```
2. Создайте и активируйте виртуальное пространство:
```bash
python3 -m venv venv
```
```bash
sourse venv/bin/activate
```
3.  Запустите приложение из корневой папки
```bash
python3 app.py
```
